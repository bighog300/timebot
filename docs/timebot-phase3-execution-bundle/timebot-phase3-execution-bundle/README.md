# Phase 3 Execution Bundle

This bundle is tailored to the current state of `timebot-main` and is intended to help Codex finish **Phase 3: Search & Intelligence** with minimal ambiguity.

## Included

- `PHASE3_GAP_REPORT.md` — repo-specific gap assessment
- `CODEX_EXECUTION_PROMPT.md` — execution prompt for Codex
- `PHASE3_WORKPLAN.md` — ordered implementation plan with acceptance criteria
- `VALIDATION_CHECKLIST.md` — completion checklist and test matrix
- `AGENTS.md` — repo-level guidance file you can drop into the repo root before running Codex

## Recommended use

1. Copy `AGENTS.md` into the repository root.
2. Start Codex at the repo root.
3. Paste `CODEX_EXECUTION_PROMPT.md` as the task.
4. Keep `PHASE3_WORKPLAN.md` open for review while Codex works.
5. Use `VALIDATION_CHECKLIST.md` before accepting the branch.

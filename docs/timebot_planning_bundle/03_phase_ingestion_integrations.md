# Phase 3: Ingestion & Integrations

## Objective
Move beyond manual upload and enable real source ingestion.

## Scope
- Connector framework
- OAuth flow
- Google Drive integration
- Gmail integration
- Background sync jobs
- Source attribution and deduplication

## Workstreams
1. Connector/auth framework
2. Google Drive ingestion
3. Gmail ingestion
4. Incremental sync engine
5. Document deduplication rules
6. Sync observability and retry policy

## Deliverables
- At least one end-to-end working connector
- Secure token handling
- Background sync with job visibility
- Source-tagged documents

## Exit criteria
- User can connect a source and ingest real documents
- Sync state is visible and reliable

# Phase 5: Insights → Actions

## Objective
Turn reporting into operational decision support.

## Scope
- Actionable dashboard
- Review triage endpoints
- Duplicate resolution workflows
- Timeline/activity improvements

## Workstreams
1. Documents needing review
2. Duplicate clusters needing action
3. Missing category / low-confidence views
4. Recent changes and activity feed
5. Dashboard UX shifts from metrics to actions

## Deliverables
- Action-oriented dashboard
- Triage views
- Clear next steps from insights

## Exit criteria
- Users can act directly from the dashboard and insight surfaces

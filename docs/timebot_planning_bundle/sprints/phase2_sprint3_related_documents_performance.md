# Phase 2 Sprint 3: Related Docs & Performance

## Goal
Make related document discovery useful and search fast enough for regular use.

## Scope
- Related docs endpoint clarity
- Performance measurement and tuning

## Tasks
- Tune relationship thresholds and return explicit relationship types
- Add pagination and limits where needed
- Measure query latency on seeded dataset
- Optimize indexes / query paths for frequent search filters

## Definition of done
- Related docs results are understandable
- Performance is acceptable on representative data

# Phase 5 Sprint 1: Actionable Dashboard

## Goal
Shift the dashboard from passive metrics to immediate work.

## Scope
- Needs review
- Missing categories
- Low-confidence items
- Recent changes

## Tasks
- Add dashboard endpoints for action queues
- Replace purely descriptive cards with action-oriented summaries
- Add tests for dashboard rollups and filters

## Definition of done
- Dashboard answers “what should I do next?”
